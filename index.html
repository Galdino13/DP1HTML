As Tags <h1> <h1/> ... <h6> <h6/>

Essa Tag serve para indicar Títulos ela vai do h1 até o h6, como por exemplo no título dessa página

A tag<p> <p>

Essa Tag serve para escrever um paragráfo como este!

A tag<mark> <mark/>

Serve para marcar algo como isso escrever no meio

A tag<small> <small/>

Já essa serve para deixar uma parte do texto pequena em comparação com o restante do paragráfo

A tag<i> <i/>

Já esta serve para deixar o texto em itálico

A tag<u> <u/>

Essa aqui serve para deixar o texto sublinhado

A tag<strong> <strong/>

Com essa o texto fica em negrito

A tag<ol> <ol/>

essa tag serve para criar listas ordenadas os itens sao criados com a tag <li> <li/>

  1. Primeiro
  2. Segundo
  3. Terceiro

A tag<ul> <ul/>

essa tag serve para criar listas não ordenadas os itens sao criados com a tag <li> <li/>

  1. Vermelho
  2. Laranja
  3. Amarelo

A tag<li> <li/>

Serve para criar os itens das listas ordenadas e não ordenadas

A tag<a> <a/>

Ela serve para direcionar para outra página ou para direcionar para uma parte da página em específica, não sei se você reparou mas no sumário se você apertar você é direcionado para a parte que explica a tag que você apertou.

A tag<hr>

Essa tag serve para criar uma linha na página como no começo depois do título e no fim do Sumário

A tag<sub> <sub/>

a sub serve para deixar o texto embaixo da linha usual do parágrafo

A tag<sup> <sup/>

a sup serve para deixar o texto acima da linha usual do parágrafo

A tag<blockquote> <blockquote/>

Agora a blockquote

normalmente é usada para citação
como você pode observar ela deixa um destaque no texto usando um recuo.

A tag<font> <font/>

A tag font foi uma das primeiras formas de aplicar estilos de fonte em elementos de texto. No entanto, ela foi considerada obsoleta e não é mais recomendada para uso. portanto não vou falar sobre ela, no lugar dela é usada a linguagem de estilização css

A tag<del> <del/>

A tag del serve para deixar o texto tachado como uma espécie de mostrar que uma parte foi retirada no texto ou para outros fins que usar necessário

A tag<abbr> <abbr/>

A tag abbr serve para você descrever uma Abreviação se passar o mouse sobre ela vai aparecer um texto explicando. usa-se o title="" e o que estiver entre as aspas explica.
